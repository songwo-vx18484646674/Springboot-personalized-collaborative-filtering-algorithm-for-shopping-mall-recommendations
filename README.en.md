# Springboot个性化协同过滤算法商城推荐

#### Description
排行榜推荐模块：查询浏览数量最多的商品，同时不包括当前登录用户浏览过的商品；个性化推荐：游客：热点推荐（根据商品总评分降序推荐）登录用户：基于用户的协同过滤推荐算法（根据购买数据），如果没有推荐结果，采用热点推荐（根据登录用户喜好标签下的商品的总购买数量降序推荐，同时是登录用户没有购买的）。相关推荐：与当前商品相同类型且收藏较多的商品，同时是当前用户没有收藏的商品。
